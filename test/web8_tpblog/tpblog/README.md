ThinkBlog 0.2
===============

#概述
一个基于ThinkPHP5的CMS系统；
ThinkBlog是我在学习新版ThinkPHP5的时候的一个作品。
目前只有一个后端管理功能，我会慢慢完善的。
希望能够和大家一起学习交流、探讨。

目前使用了ThinkPHP5框架、kindeditor编辑器、icomoon图标、layer弹层、DataTables、dropzone文件上传等框架组件；

**目前已经有的功能：**

* 文章管理；
* 分类管理；
* 链接管理；
* 配置项管理；
* 用户信息修改；
* 密码修改；
